감성 분석의 기초

텍스트에 담긴 감정을 읽어내는 방법

감성 분석이란?

텍스트에 담긴 감정의 방향(긍정/부정/중립)과 강도를 파악하는 분석 방법입니다. 단순한 분류를 넘어, 글쓴이의 태도와 감정 상태까지 읽어냅니다.

감성 분석의 3단계

  • Level 1 — 극성 분류: 긍정/부정/중립 (가장 기본)
  • Level 2 — 감정 라벨링: 기쁨/분노/슬픔/불안/놀라움/기대 등 세분화
  • Level 3 — 관점 기반 감성: "수업 내용은 좋았는데(+) 과제가 너무 많았다(-)" 처럼 항목별 감성 분리

비교: 좋은 vs 나쁜 프롬프트

❌ 부족한 프롬프트 
이 텍스트의 감성을 분석해줘
✅ 좋은 프롬프트 
너는 교육 평가 전문가야.
아래 강의평가 응답에서 감성을 분석해줘.

분석 기준:
- 극성: 긍정(+) / 부정(-) / 중립(0) / 혼합(±)
- 감정 라벨: 만족/기대/성취감/불만/불안 중 택1
- 강도: 1(약) ~ 5(강)
- 관련 주제: 수업내용/교수법/평가/환경

"교수님 설명이 정말 이해하기 쉬워서 기대돼요" →
극성: +, 감정: 만족/기대, 강도: 5, 주제: 교수법
⚠️ 감성 분석은 개인의 감정을 다루므로 윤리적 주의가 필요합니다. 개인을 판단하는 도구가 아닌, 전체 패턴을 파악하는 도구로 활용하세요.

감성 사전 구축

교육 맥락에 맞는 감성 사전 만들기

교육 맥락의 감성 사전이 필요한 이유

  • 일반 감성 사전은 "어렵다"를 부정으로 분류하지만, 교육 맥락에서는 "어렵지만 성장했다"는 긍정일 수 있습니다.
  • 교육 현장 특유의 표현("멘붕", "꿀잼", "갓교수", "노잼" 등)은 일반 사전에 없습니다.
  • 커스텀 감성 사전을 만들면 분석 정확도가 크게 올라갑니다.
너는 교육학 전문가이자 텍스트 분석가야.

[교육 현장]에서 사용되는 감성 표현을 정리해줘.

다음 표 형식으로 작성해줘:

| 표현 | 극성(+/-/중립) | 감정 라벨 | 강도(1~5) | 맥락 설명 |

[포함할 범주]
1. 수업 내용에 대한 감성 표현 (10개)
2. 교수법에 대한 감성 표현 (10개)
3. 평가/과제에 대한 감성 표현 (10개)
4. 학습 경험에 대한 감성 표현 (10개)

[교육 현장 맥락]

감성 분석 실습

프롬프트 설계와 실행

감성 분석 프롬프트 설계

3주차의 프롬프트 기법 + 4주차의 분류 기법 + 감성 사전을 조합하여 강력한 분석 프롬프트를 만듭니다.

너는 [직함]이자 감성 분석 전문가야.

아래 텍스트에서 감성을 분석해줘.

[분석 기준]
- 극성: 긍정(+) / 부정(-) / 중립(0) / 혼합(±)
- 감정 라벨: 만족/기대/성취감/불만/불안/좌절/무관심 중 택 1~2
- 강도: 1(약) ~ 5(강)
- 관련 주제: 수업내용/교수법/평가/환경/기타

[감성 판단 참고]
- "어렵다" → 맥락 확인 필요 (부정적 불만 vs 긍정적 도전)
- "괜찮다" → 중립 (적극적 긍정 아님)
- "~하면 좋겠다" → 부정(현 상태 불만) + 제안

[출력 형식]
| 번호 | 원문(요약) | 극성 | 감정 라벨 | 강도 | 관련 주제 | 판단 근거 |

[Few-shot 예시]
"교수님 설명이 정말 이해하기 쉬워서 수업이 기다려졌어요" → +, 만족/기대, 5, 교수법, "정말 이해하기 쉬워서"+"기다려졌어요"가 강한 긍정

"과제가 많아서 힘들었지만, 실력이 늘었다는 건 느꼈어요" → ±, 좌절/성취감, 3, 평가, 부정(과제 부담)+긍정(성장) 혼합

[분석 대상]
(텍스트를 여기에 붙여넣으세요. 최소 5건 이상의 응답)
💡 핵심: 감성 분석의 '혼합' 범주가 중요합니다. 교육 현장의 텍스트는 '힘들었지만 좋았다' 같은 복합 감정이 많습니다.

해석과 시각화

결과에서 인사이트 도출하기

분석 결과를 해석하는 3가지 관점

  • 분포 관점: 긍정/부정/중립의 비율은? → "70% 긍정이면 양호, 30% 이상 부정이면 주의"
  • 주제별 관점: 어떤 주제에서 부정이 집중되는가? → "교수법은 긍정적인데 평가에서 부정이 많다"
  • 변화 관점: 시간에 따라 감성이 어떻게 변하는가? → "학기 초 불안 → 중반 적응 → 말 만족"
너는 교육 데이터 분석 전문가야.

아래 감성 분석 결과를 해석하고, 교수자를 위한 인사이트를 도출해줘.

[분석해줄 것]
1. 전체 감성 분포 (긍정/부정/중립/혼합 비율)
2. 주제별 감성 비교 (어떤 주제가 가장 긍정적/부정적인지)
3. 주요 불만 사항 TOP 3과 개선 제안
4. 주요 강점 TOP 3과 유지 방안

[출력 형식]
- 각 항목별로 구조화하여 작성
- 숫자/비율을 반드시 포함
- 교수자가 바로 행동할 수 있는 구체적 제안 포함

[감성 분석 결과]
(여기에 실습 3의 결과를 붙여넣으세요)
