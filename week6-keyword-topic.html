5주차 복습: 혼합 감성, 어떻게 구분할까?

지난 주 어려웠던 혼합 감성 판단 기준을 정리합니다

💬 지난 수업 피드백

"감성 분석에서 혼합 감성이 무슨 말인지 잘 모르겠다." "긍정인지 혼합인지 기준을 어떻게 세워야 하는지 어려웠다."

이번 섹션에서 혼합 감성의 판단 기준을 명확하게 정리하고 넘어가겠습니다.

핵심 원리: "접속사가 방향을 바꾸는가?"

혼합 감성의 핵심은 하나의 문장 안에서 감정의 방향이 전환되는 것입니다. 단순히 강도가 약한 긍정이나 애매한 표현은 혼합이 아닙니다.

🚦 4가지 판단 기준 (의사결정 트리)

긍정 + 감정 방향이 하나이고, 그 방향이 긍정
"교수님 설명이 정말 이해하기 쉬웠어요" → 긍정적 감정만 존재
부정 − 감정 방향이 하나이고, 그 방향이 부정
"과제가 너무 많아서 힘들었어요" → 부정적 감정만 존재
혼합 ± 한 문장 안에 긍정 요소와 부정 요소가 동시에 존재
"과제가 많아서 힘들었지만, 실력이 늘었다는 건 느꼈어요"
👉 "힘들었다"(부정) + "실력이 늘었다"(긍정) = 두 방향 공존
중립 0 감정 방향이 드러나지 않거나 사실 진술에 가까움
"이번 학기 수업은 월요일 3교시입니다" → 감정 없음
💡 혼합을 가르는 핵심 신호: "~지만", "~하지만", "~에도 불구하고", "그래도", "반면에" 같은 역접 접속사가 있고, 접속사 앞뒤의 감정 방향이 다르면 혼합입니다.
문장판정근거
"수업은 괜찮았어요" 중립 적극적 긍정이 아님. "괜찮다"는 소극적 수용 → 중립 처리
"어려웠지만 따라갈 만했어요" 혼합 ± "어려웠다"(부정) + "따라갈 만하다"(소극적 긍정) → 방향 전환
"내용은 좋은데 시간이 부족해요" 혼합 ± "좋다"(긍정) + "부족하다"(부정) → 주제별 감정이 다름
"처음엔 어려웠는데 나중엔 재밌었어요" 혼합 ± 시간에 따라 감정 변화 → 부정에서 긍정으로 전환
"진짜 너무 어려워요" 부정 − "어렵다"만 있고 긍정 요소 없음 → 단일 부정. 강도가 센 부정이 혼합은 아님!
⚠️ 주의: "매우 부정적"과 "혼합"은 다릅니다. 강도가 세다고 혼합이 되는 게 아니라, 방향이 두 개 이상이어야 혼합입니다.

키워드 추출의 기초

텍스트에서 핵심 단어를 골라낸다

키워드 추출이란?

텍스트 덩어리에서 내용을 대표하는 핵심 단어나 구를 자동으로 뽑아내는 기술입니다. 사람이 하면 수백 건의 글을 읽고 "자주 나오는 말, 중요한 말"을 정리하는 작업을 AI가 대신합니다.

키워드 추출의 3가지 접근법

  • 빈도 기반: 자주 등장하는 단어 = 중요한 단어. 가장 직관적
  • TF-IDF 기반: "이 문서에서는 자주 나오지만 다른 문서에서는 안 나오는" 단어가 더 중요. 차별성 반영
  • AI 기반: 문맥을 이해하고 핵심 개념을 직접 추출. 가장 유연

TF-IDF = "이 글에서만 특별한 단어"를 찾는 방법

TF (Term Frequency): 이 문서에서 얼마나 자주 나오나? → 많이 나올수록 높음

IDF (Inverse Document Frequency): 전체 문서에서 얼마나 드물게 나오나? → 드물수록 높음

예시: "수업"이라는 단어는 강의평가 50건에 모두 나옴 → IDF 낮음 (별로 특별하지 않음). "시뮬레이션"은 3건에만 나옴 → IDF 높음 (그 수업의 특징!)

AI한테 TF-IDF를 직접 계산하라고 할 필요 없이, "문맥을 고려해서 핵심 키워드를 뽑아줘"라고 하면 비슷한 결과를 얻을 수 있습니다.

💡 4주차 분류 vs 6주차 키워드 추출: 분류는 "이 글이 어떤 카테고리에 속하는가?" (= 라벨 붙이기), 키워드 추출은 "이 글에서 중요한 단어가 무엇인가?" (= 핵심어 뽑기)입니다. 관점이 다릅니다!

키워드 추출과 빈도 분석

AI로 핵심 단어를 뽑고 빈도를 세다

키워드 추출 프롬프트 설계

3주차 프롬프트 기법 + 분석 목적을 조합하여 정밀한 키워드 추출 프롬프트를 만듭니다.

너는 교육 데이터 분석 전문가이자 텍스트 마이닝 전문가야.

아래 텍스트에서 키워드를 추출하고 빈도를 분석해줘.

[분석 기준]
- 키워드 유형: 단일어(1어절) + 복합어(2~3어절 구)
- 불용어 제외: 조사, 접속사, 일반적 동사(하다, 있다, 되다 등)
- 최소 등장 횟수: 2회 이상

[출력 형식]
| 순위 | 키워드 | 빈도 | 유형(단일/복합) | 관련 맥락 요약 |

상위 15개를 빈도 내림차순으로 정리해줘.

추가로, 아래 항목도 분석해줘:
1. 가장 자주 함께 등장하는 키워드 쌍 (공출현 TOP 5)
2. 긍정 맥락에서 자주 등장하는 키워드 vs 부정 맥락에서 자주 등장하는 키워드

[분석 대상]
(텍스트를 여기에 붙여넣으세요. 최소 10건 이상 권장)

비교: 좋은 vs 나쁜 프롬프트

❌ 나쁜 프롬프트 
"이 글에서 키워드 뽑아줘"
✅ 좋은 프롬프트 
"교육 현장 텍스트 분석 전문가로서, 아래 강의평가 50건에서
키워드(단일어+복합어)를 추출하고 빈도를 표로 정리해줘.
불용어는 제외하고, 최소 2회 이상 등장한 키워드만 포함.
긍정/부정 맥락별로도 구분해줘."
💡 핵심: "불용어 제외" 지시가 없으면 "~하다", "~있다" 같은 의미 없는 단어가 상위에 올라옵니다. 반드시 명시하세요.

주제 클러스터링

비슷한 키워드끼리 묶어 숨은 주제를 발견한다

주제 클러스터링이란?

추출된 키워드들을 의미적 유사성에 따라 그룹으로 묶는 작업입니다. 개별 키워드만으로는 보이지 않던 상위 주제(테마)를 발견할 수 있습니다.

교육 데이터 주제 클러스터링 예시

강의평가 키워드 30개를 AI로 클러스터링한 결과:

주제(클러스터)포함 키워드비중
수업 방법 실습, 활동, 토론, 발표, 팀플, 참여 35%
학습 부담 과제, 시험, 분량, 시간, 어렵다, 벅차다 28%
교수 역량 설명, 피드백, 소통, 열정, 이해 22%
학습 환경 강의실, 시설, 교재, 온라인, 장비 15% 
너는 질적 연구 방법론 전문가야.

아래 키워드 목록(빈도 포함)을 의미적 유사성에 따라 3~5개 주제(클러스터)로 묶어줘.

[클러스터링 기준]
- 의미적으로 관련된 키워드를 하나의 주제로 묶기
- 각 주제에 대표 이름(한국어 2~4단어) 부여
- 주제별 포함 키워드, 키워드 수, 전체 대비 비중(%) 명시

[출력 형식]
| 주제명 | 포함 키워드 | 키워드 수 | 비중(%) | 핵심 해석 |

추가 분석:
1. 가장 비중이 큰 주제에 대한 상세 해석 (3줄)
2. 주제 간 연결 관계가 있다면 설명
3. 교육 현장에서의 시사점

[키워드 목록]

교육 현장 적용

키워드와 주제 분석 결과를 수업 개선에 연결한다

분석 결과를 행동으로 바꾸는 3단계

  • 1단계 — 발견: "학생들이 가장 많이 이야기하는 주제는 무엇인가?"
  • 2단계 — 해석: "그 주제에서 긍정인가 부정인가? 왜 그런가?"
  • 3단계 — 행동: "다음 학기에 구체적으로 무엇을 바꿀 것인가?"
너는 교육과정 개선 컨설턴트야.

아래 키워드 추출 및 주제 클러스터링 결과를 보고,
교수자가 수업 개선에 활용할 수 있는 시사점을 도출해줘.

[분석해줄 것]
1. 학생 관심도 TOP 3 주제 (가장 많이 언급된 주제)
2. 각 주제의 감성 경향 (5주차 분석 결과와 연결)
3. 즉시 개선 가능한 항목 (다음 수업부터 적용 가능)
4. 장기적 개선이 필요한 항목 (교육과정 차원)
5. 학생 니즈와 현행 수업의 갭 분석

[출력 형식]
