분류의 기초

좋은 분류 체계를 설계한다

텍스트 분류란?

비정형 텍스트를 미리 정의한 카테고리에 배정하는 과정입니다. 베럴슨의 '체계적·객관적' 내용분석 원칙과 직결됩니다.

좋은 분류 체계의 조건 3가지

  • 상호배타성(Mutually Exclusive): 하나의 텍스트는 하나의 범주에만 속해야 한다
  • 포괄성(Exhaustive): 모든 텍스트가 어딘가에 분류되어야 한다
  • 일관성(Consistent): 누가 분류해도 같은 결과가 나와야 한다
분류 대상카테고리 예시활용 목적
강의평가 서술형수업내용/교수법/평가/환경/기타수업 개선 우선순위 파악
학생 성찰일지인지적/정서적/행동적/사회적학습 경험 패턴 분석
학부모 상담 기록학업/생활/진로/교우관계/가정환경상담 주제 트렌드 파악
자유 서술 응답긍정/부정/중립/제안만족도 및 요구 파악
생기부 특기사항교과역량/인성/진로/특기학생 프로필 자동 정리
💡 분류 체계는 연구 질문에 따라 달라집니다. "무엇을 알고 싶은가?"를 먼저 정하세요.

분류 프롬프트 설계

AI에게 분류 기준을 명확하게 알려준다

분류 프롬프트의 4가지 핵심 요소

  • ① 역할 지정: AI에게 분류 전문가 역할 부여
  • ② 분류 기준 명시: 카테고리 목록 + 각 카테고리의 정의
  • ③ 출력 형식: 분류 결과의 구조 (표, JSON 등)
  • ④ Few-shot 예시: 정확한 분류의 예시 2~3개 제공

비교: 기준 없음 vs 기준 있음

❌ 기준 없음 
이 텍스트들을 분류해줘
✅ 완성된 분류 프롬프트 
너는 교육 내용 분석 전문가야.

아래 텍스트를 다음 카테고리로 분류해줘:
1. 수업 내용 — 교과 내용의 난이도, 분량, 흥미도
2. 교수 방법 — 강의 방식, 활동, 매체 활용
3. 평가 — 시험, 과제, 피드백

분류 규칙:
- 하나의 텍스트는 가장 관련 높은 카테고리 1개에만 배정
- 판단이 어려운 경우 "기타"로 분류하고 이유를 메모

분류 예시:
예시 1: "교수님 설명이 너무 빨라서 따라가기 어려웠어요" → 교수 방법
예시 2: "과제가 너무 많아요" → 평가

[텍스트를 여기에 붙여넣으세요]
✅ 3주차에 배운 4가지 기법(역할, 형식, 체이닝, Few-shot)이 여기서 모두 활용됩니다!
